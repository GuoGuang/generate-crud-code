# generate_crud_code
根据实体类生成原始代码

在带有@Entity注解的实体类上右键单击"Generator CRUD"，然后将生成以下模块:
- Controller REST API with Swagger DOC API
- DAO(JpaRepository)
- DTO
- Service (with page query)
- Mapper mapstruct
- Mockito Junit5 Unit Test
